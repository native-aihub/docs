---
title: "Get Extraction"
description: "Get a specific extraction by ID"
---

## Endpoint

```
GET https://api.documind.cloud/api/v1/data/extractions/{extraction_id}
```

## Path Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `extraction_id` | string (UUID) | Yes | ID of the extraction |

## Response

```json
{
  "id": "extraction-id",
  "document_id": "doc-id",
  "results": {...},
  "reviewed_results": {...},
  "needs_review": false,
  "is_reviewed": false,
  "created_at": "2024-01-15T10:30:00Z"
}
```
